💡 O Projeto Real

Um grafo interativo de competências que analisa automaticamente o ecossistema de um time/empresa e revela quem sabe o quê. Ideal para:

  • Gestores: monte squads balanceados em 1 clique — "quem sabe Rust + AWS?"
  • RH/Tech Leads: identifique gaps de skills e planeje contratações com dados
  • Devs: descubra colegas que dominam tecnologias que você quer aprender
  • Consultorias: apresente a capacitação do time para clientes de forma visual

🔒 Limitações das Plataformas

As APIs públicas impõem restrições reais que impactam o escopo do projeto:

PlataformaLimitaçãoImpactoStatus
GitHub 60 req/h sem token · 5000 req/h com PAT Viável com autenticação — 1 token cobre todo o time ✅ Viável
GitHub Events API: apenas últimos 90 dias de atividade Histórico profundo requer Git clone local (mais lento) ⚠️ Limitado
GitHub Search API: max 1000 resultados por query OK para times < 1000 devs ✅ Viável
LinkedIn API de skills removida em 2023 Não é mais possível extrair skills do perfil via API oficial 🚫 Bloqueado
LinkedIn API Partners apenas para produtos aprovados Sem acesso a dados de outros usuários sem contrato enterprise 🚫 Bloqueado
LinkedIn Rate limit: ~100 req/dia por app Inviável para times grandes 🚫 Bloqueado
✅ Estratégia adotada:
  • GitHub como fonte primária — linguagens, commits, estrelas, contribuições
  • Auto-reporte como complemento — formulário interno para skills não detectáveis (soft skills, metodologias)
  • LLM para inferir skills a partir de descrições de repo e README
  • LangGraph como orquestrador do pipeline de extração → normalização → grafo
